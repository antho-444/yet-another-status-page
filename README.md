## Quick Start

```bash
# Clone the repository
git clone https://github.com/Hostzero-GmbH/yet-another-status-page.git
cd yet-another-status-page

# Start the services
docker compose up -d
```

Visit:

- **Status Page**: http://localhost:3000
- **Admin Panel**: http://localhost:3000/admin
